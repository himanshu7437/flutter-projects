# Flutter Learning

This repository contains Flutter projects and code samples created during my Flutter learning journey.

## Getting Started

To run any project in this repository:

1. Make sure Flutter is installed.  
2. Clone this repository.  
3. Navigate to the project folder.  
4. Run `flutter pub get` to fetch dependencies.  
5. Use `flutter run` to launch the app.

## Folder Structure

- `lib/` — Main Dart source files.
- `test/` — Unit and widget tests.
- `extrafiles/` — Ignored in `.gitignore`.
